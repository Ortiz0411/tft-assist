# TFT Assist


Desktop app built in Python. Based on heuristics to assist the player to make decisions in TeamFight Tactics (TFT)

## Project goals
Analize the game state and recommend:
- Best compositions
- Priority in champions and items
- Strategy (reroll, fast 8, fast 9, more.)
- Champions positioning

The app is data-driven (Data Dragon), refresh each patch.

## Tech stack
- Python
- Pyside6 (Qt)
- JSON data
